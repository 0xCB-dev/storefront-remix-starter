# Vendure Remix Storefront Starter

A storefront for [Vendure](https://www.vendure.io) and [Remix](https://remix.run).

## Development

1. Create an .env file in the root:
   ```
   ENCRYPTION_KEY=abc
   VENDURE_API_URL=http://localhost:3001/shop-api
   ```
2. npm run dev
